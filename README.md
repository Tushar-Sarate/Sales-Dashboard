# 📊 Sales Analytics Dashboard

## Overview
Developed an interactive Sales Analytics Dashboard using **PostgreSQL** and **Power BI** to analyze sales performance, profitability, customer behavior, and business trends.

## Tools & Technologies
- PostgreSQL
- SQL
- Power BI
- Excel

## Key Features
- Revenue and Profit Analysis
- Sales Trend Visualization
- Country & Category Performance
- Customer Demographic Analysis
- Sales Representative Performance Tracking

## Key Insights
- Identified top-performing products and regions.
- Analyzed customer purchasing patterns.
- Evaluated the impact of discounts on profitability.
- Tracked monthly sales and profit trends.
